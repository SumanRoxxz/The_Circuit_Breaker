WHAT WE BUILD

ESP32 WIFI CAR

EMBEDDED SYSTEMS

Fully autonomous WiFi-controlled car powered by ESP32. Features real-time video streaming, obstacle avoidance via ultrasonic sensors, and a mobile dashboard UI over WebSocket.

ESP32 WebSocket C++ WiFi
🤖

AUTONOMOUS DRONE

ROBOTICS / AI

GPS-waypoint drone with AI-powered obstacle avoidance using OpenCV and a Raspberry Pi 4. PID-tuned flight controller, real-time telemetry, and live FPV video feed.

RPi4 OpenCV PID GPS
📡

FUNCTION GENERATOR

ANALOG ELECTRONICS

DDS-based multi-waveform function generator capable of producing sine, square, triangle, and sawtooth waves up to 20MHz. Custom PCB design with OLED display and rotary encoder UI.

DDS PCB OLED STM32
🦾

EMG ROBOTIC ARM

BIO-ELECTRONICS

Muscle-signal-controlled prosthetic arm reading EMG signals via surface electrodes. 6-DOF servo system with real-time signal classification using edge ML on microcontroller.

EMG TinyML 6-DOF RTOS
